# MIO PRIME v1.2

**Owner:** Chayan Khatua
**System:** Autonomous AI Agent Framework

## Overview
MIO PRIME is a Python-based agent designed to assist its owner with various tasks, including device control and information retrieval.

## Features
- **Owner Authentication:** Voice phrase ("Mio, proceed.") and token-based verification.
- **Device Control:** Open applications.
- **Memory:** Secure logging of activities and conversation history.
- **Extensible Skills:** Modular design for adding new capabilities.

## Usage
1.  Navigate to the `mio_prime` directory.
2.  Run `python main.py`.
3.  Authenticate when prompted.
4.  Issue commands like "open calculator" or "who are you".

## Security
- Access is strictly limited to the owner.
- All actions are logged.
